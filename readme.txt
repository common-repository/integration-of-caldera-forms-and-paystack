Caldera Forms Paystack Integration

== Description ==

The integration of Caldera Forms and Paystack plugin lets you add a new payment option Processor to Caldera form. It automatically send data from your Caldera form to your Paystack when the form is submitted.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Integration of Caldera Forms and Paystack".
3. Click install.
4. Click activate.
